Introduction

The goal of this project is to deepen your knowledge by using Kubernetes to virtualize a network and to make you discover what a production environment really looks like. You will do what is called "clusturing".

General instructions

  • You have to make all the files necessary for the configuration of your application in an srcs folder.
  • Your setup.sh file should be at the root of your directory and should allow you to set up your entire application.
  • This subject requires you to put into practice a lot of notions that you have already covered for some, for others not. So we advise you not to be afraid to read a lot of documentation about the use of Kubernetes, as well as about everything you will need to solve the project.

Just to keep things simple, the login credentials for the different services will always be the very original admin/admin pair.
